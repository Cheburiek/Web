How to use calculator
  • "+" - just add two numbers, for example: 10+5
  • "-" - just subtract two numbers, for example: 10-5
  • "*" - just multiply two numbers, for example: 10*5
  • "/" - just divide two numbers, for example: 10/5
  • "pow" - use this operator **, for example: 10**5
  • "sqrt" - use this operator **, for example: 25**(1/2)
